School Management System.
It helps to manage students,teachers,courses,groups.i have done it because i wanted to try in practice what i learned.
Used technologies:Java,Spring Boot,MVC,Data JPA,Security,Mail Sender.
There are two roles:ADMIN,USER.ADMIN can manage everything at school , USER is student.
